Memory House Cafe สามพราน
รายละเอียดเกี่ยวกับค่าเฟ่

จุดเด่น

Memory House Cafe สามพราน คาเฟ่เปิดใหม่ที่เป็นที่ฮอตฮิตของหนุ่มสาวสายโซเชียลที่พากับมาแชะภาพถ่ายรูปอัพลงไอจีกันเป็นอย่างมากและด้วยความที่ทำเลของคาเฟ่ติดริมแม่น้ำทำให้บรรยากาศของคาเฟ่เย็นสบาย ไม่ร้อนมากและมีหลายโซนไม่ว่าจะเป็นโซนริมน้ำที่มีความเป็นส่วนตัว โซนปิกนิกที่มีผ้าปูและพร็อพอำนวยความสะดวกอย่างครบครันหรือว่าจะเป็นโซนห้องแอร์ที่สำหรับวันไหนที่อากาศร้อนก็สามารถไปหลบแดด นั่งทานขนมอร่อยๆได้เลยค่ะ อีกทั้งตัวคาเฟ่ก็มีการเปิดเพลงเกาหลีคลอเบาๆทำให้ได้บรรยากาสความเป็นคาเฟ่สไตล์เกาหลีสุดๆเลย สำหรับใครที่อยากแวะมาทานขนมอร่อยๆและเก็บภาพสวยๆอย่าลืมแวะมาที่นี่กันนะคะ
